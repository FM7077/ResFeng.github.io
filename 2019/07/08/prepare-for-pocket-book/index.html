记账本准备工作

发表于 2019-07-08   |   更新于 2019-07-11 | 分类于 笔记 |
字数统计: 422

关于记账本 app 的一些想法和准备工作

设想

一个最基本功能是记流水账, 附加功能是小金库记录功能的记账本.

小金库即各种基础现金源, 包括现金, 银行卡, 储蓄账户等, 不包括电话卡, 公交卡等.

流水账包括出和入的, 每一笔都需要人工录入, 目前设想参数:

参数 功能 / 备注
id id
现金源id 完成这笔流水的现金源
金额 通过正负分辨进出
备注 备注
时间

现金源流水:

每次产生流水都会自动生成现金源流水, 现金源流水在查看现金源资金情况的时候展示

参数 功能 / 备注
id id
现金源id 现金源id
金额 通过正负分辨进出
对应流水id
备注 备注
时间

功能

  1. 流水账记录 CRUD
  2. 现金源记录 CRUD
  3. 流水折线图
  4. 数据导出 csv

难点

现金源之间的转账如果涉及到手续费如何记录:

为现金源之间的转账设立专门的入口, 用户手动选择两个现金源, 填写转账金额, 手续费, 确认后提交两条流水, 现金源为转出的现金源, 金额为手续费, 备注为”现金源1转账xx金额至现金源2”. 同时分别为现金源1, 2生成两条流水. 然后再从现金源1中减去转账金额和手续费, 现金源2中增加转账金额.

若用户修改现金源金额如何记录:

直接记录流水, 自动生成金额, 备注为”用户修改”
