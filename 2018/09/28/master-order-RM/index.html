师傅单据RM

发表于 2018-09-28   |   更新于 2018-11-07 | 分类于 笔记 |
字数统计: 1.1k

关于这个程序的一些脑洞

开发工具

  1. BS架构
  2. Idea
  3. MySQL 或 SQLServer
  4. 脚架 dp-pro

需求分析

  1. 已有师傅单据程序所有功能
  2. 货物分类管理
  3. 师傅小程序下单功能
    1. 登录功能 微信小程序登录流程控制
    2. 检索货物, 提交购物单功能
    3. 在本程序接收校验购物单, 并转换为实际单据
  4. 单据推送( 用以校对出货 ), 同样是通过小程序实现
    1. 与师傅小程序下单类似而精简的登录功能
    2. 只需实现简单推送, 用以实现校对出货即可
  5. 日结, 月结功能
  6. 程序内实现分页查询
  7. 自动提醒上一次购买的价格
  8. 根据实际情况看是否要实现单位转换

库结构设计

货物库( goods )

列名 类型 属性 备注
good_id varchar(7) pk g+6位序列号, 由程序实现自增
good_name varchar(100) not null 货物品名
good_unit varchar(6) not null 单位
brand_id varchar(5) not null 所属品牌 id
factory_price DECIMAL( 6, 2 ) 默认 0 厂价
purchase_price DECIMAL( 6, 2 ) 默认 0 进货价
wholesale_price DECIMAL( 6, 2 ) 默认 0 批发价
retail_price DECIMAL( 6, 2 ) 默认 0 零售价
stock DECIMAL( 6, 2 ) 默认 0 库存

历史价格库( price_history )

列名 类型 属性 备注
price_id varchar(10) pk 货物id+3位序列号
good_id varchar(7) not null 货物id
factory_price DECIMAL( 6, 2 ) not null 厂价
purchase_price DECIMAL( 6, 2 ) not null 进货价
wholesale_price DECIMAL( 6, 2 ) not null 批发价
retail_price DECIMAL( 6, 2 ) not null 零售价
gmt_create varchar(10) not null 创建时间
gmr_modified varchar(10) not null 修改时间, 默认与创建时间同日

品牌类型库( brand )

列名 类型 属性 备注
id varchar(6) 主键 6位序列号
parent_id varchar(6) not null 父id, 类的父id 为0
name varchar(10) not null 类名/品牌名

单据库( bill_title )

列名 类型 属性 备注
bill_id varchar(13) pk 门店代号(目前初始为a) + 6位年月日 + 6位序列号
gmr_create varchar(10) not null 创建时间 年-月-日
gmr_modified varchar(10) not null 修改时间 年-月-日, 默认与创建时间同日
customer_id varchar(7) not null 顾客 id 若是已存于顾客库的直接引用顾客库的 id, 若是零售单则自建一个序列号: tc + 8位序列号(该序列号根据 开头为 tc 的数量实现自增)
bill_status int 默认0 0: 出库单, 1: 退货的
place_id varchar(10) 工地 id

单据详细库( bill_detail )

列名 类型 属性 备注
bdetail_id varchar(16) pk 所属单据 id + 3位序列号
bill_id varchar(13) not null 所属单据 id
good_id varchar(7) not null 货物 id
good_name varchar(100) not null 货物名称
good_unit varchar(6) not null 单位
factory_price DECIMAL( 6, 2 ) not null 厂价
purchase_price DECIMAL( 6, 2 ) not null 进货价
wholesale_price DECIMAL( 6, 2 ) not null 批发价
retail_price DECIMAL( 6, 2 ) not null 零售价
num DECIMAL( 6, 2 ) not null 数量

顾客库

列名 类型 属性 备注
customer_id varchar(7) pk ct + 5位序列号
customer_name varchar(20) not null 顾客名称
customer_phone varchar(11) 顾客手机号( 可为空 )
union_id varchar(28) 通过微信注册的用户的 union id

工地库

列名 类型 属性 备注
place_id varchar(10) pk cp + 8位序列号
customer_id varchar(7) not null 所属顾客 id
place_name varchar(20) not null 工地名称
place_address varchar(100) 工地地址
place_status int 默认 0 0: 未完工, 1: 已完工

临时顾客库

列名 类型 属性 备注
union_id varchar(28) pk 微信的 union id
customer_name varchar(20) not null 顾客名称
customer_phone varchar(11) 顾客手机号( 可为空 )

订单库

列名 类型 属性 备注
order_id varchar(13) pk o + 年月日+6位序列号
gmr_create varchar(10) not null 创建时间 年-月-日
place_id varchar(10) not null 工地 id
order_status int 默认 0 0: 已录入, 1: 未录入

订单详细库

列名 类型 属性 备注
odetail_id varchar(16) pk 所属订单 id + 3位序列号
order_id varchar(13) not null 所属订单 id
good_id varchar(7) not null 货物 id
good_name varchar(100) not null 货物名称
num DECIMAL( 6, 2 ) not null 数量

微信推送单据

列名 类型 属性 备注
bill_id varchar(13) pk 单据 id
